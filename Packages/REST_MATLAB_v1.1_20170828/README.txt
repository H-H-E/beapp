This GUI is used to do the REST transformation processing for EEG data.The detailed procedures for REST transformation processing is descripted as follow,
	
	[1]. Calculate leadfield matrix;

	[2]. Import data;

	[3]. Plot data (optional);

	[4]. Exclude channels (e.g., EOG, EKG, and bad channels, etc);

	[5]. Import corresponding leadfield matrix;

	[6]. Run REST & export data;

	[7]. Clear dataset(s) (optioanl);
